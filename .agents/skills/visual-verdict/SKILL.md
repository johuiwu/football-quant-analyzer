### 适用场景
- 前端页面开发，需与设计稿比对
- UI 组件、海报、排版样式还原
- 样式适配与视觉一致性校验
### 核心职责
- 对照参考截图/设计稿，比对整体布局、层级、间距
- 校验色彩色值、元素尺寸、字体样式、圆角、阴影等细节
- 输出偏差清单，并给出可直接落地的 CSS/样式调整方案
- 最终实现与参考图高度一致
### 触发关键词
`对比效果图` `校验样式` `还原界面` `排版检查` `视觉比对` `UI对齐` `核对页面布局` `校准色彩尺寸` `修正视觉偏差` `复刻设计样式`
### 工作规范（执行流程）
1. 获取参考图或设计规范链接，若用户未提供则提示提供。
2. 逐区域比对结构、色彩、字体等属性，记录所有差异点。
3. 生成偏差报告，按严重程度排序，并附带修正代码。
4. 对修正后结果进行二次比对，确认无视觉偏差。
